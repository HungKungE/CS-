## Stack & Queue & 힙

1. 스택 ( Stack )
- 후입선출 ( LIFO )
- 프링글스 생각하면댐!
- 구현 자료형은 배열, 리스트
- ex ) 메모리의 함수 호출, 로컬변수 관리 ( 정적 )

2. 큐 ( Stack )
- 선입선출 ( FIFO )
- 빨대 생각하면댐!
- 구현 자료형은 배열, 리스트
- ex) BFS문제를 풀 때 탐색 순서 관리함.

3. 힙 ( Heap )
- 메모리의 동적인 영역.
- 사용자가 동적으로 할당한 메모리를 해제하지 않으면 메모리 누수 발생
- 직접 해제 , GC 사용 등으로 메모리 해제함.
