## Stack & Queue & 힙

1. 스택 ( Stack )
- 후입선출 ( LIFO )
- 프링글스 생각하면댐!
- 구현 자료형은 배열, 리스트
- ex ) 메모리의 함수 호출, 로컬변수 관리 ( 정적 )

2. 큐 ( Stack )
- 선입선출 ( FIFO )
- 빨대 생각하면댐!
- 구현 자료형은 배열, 리스트
- ex) BFS문제를 풀 때 탐색 순서 관리함.

3. 힙 ( Heap )
- 최대값, 최소값을 빠르게 찾기 위한 이진트리.
- 최소 힙 ( 부모 < 자식 ), 최대 힙 ( 부모 > 자식 )
- 시간 복잡도는 깊이에 비례 : O(logN)
